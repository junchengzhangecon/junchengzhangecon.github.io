# Juncheng Zhang Academic Website

This is the source code for Juncheng Zhang's personal academic website, built with [Quarto](https://quarto.org/) and adapted from [Dr. Gang He's Quarto Academic Website Template](https://github.com/drganghe/quarto-academic-website-template).

## Local preview

```bash
quarto preview
```

## Build

```bash
quarto render
```

The rendered website is written to the `docs/` folder for GitHub Pages.

## Deployment target

The intended GitHub Pages repository name is:

```text
junchengzhangecon.github.io
```

After pushing to GitHub, set Pages to publish from the `main` branch and the `/docs` folder.
